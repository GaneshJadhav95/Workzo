Workzo – Freelancing Marketplace Website

📌 Project Overview

Workzo is a freelancing marketplace platform where clients can post jobs and freelancers can find work. The platform connects clients and freelancers in one place to collaborate on projects, send proposals, and communicate easily.

This project includes separate dashboards for **Freelancers**, **Clients**, and **Admin** to manage the platform efficiently.

🚀 Features

Freelancer Side

* Create and manage profile
* Find available jobs
* Send proposals to clients
* Freelancer dashboard
* Messaging system with clients
* Account settings and profile management

Client Side

* Create and manage client profile
* Post new jobs
* View proposals from freelancers
* Client dashboard
* Messaging with freelancers
* Manage uploaded jobs

Admin Panel

* Admin dashboard overview
* View total users and active users
* User management table
* Monitor platform activity
